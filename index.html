Règlements du jeu

Le but du jeu POKER est d'obtenir une combinaison de carte faute de quoi, le joueur perd sa mise. Les combinaisons possibles sont les suivantes :

  • une paire : 2 cartes de même valeur
  • Deux paires : 2 x 2 cartes de même valeur
  • Brelan : 3 cartes de même valeur
  • Quinte : 5 cartes de même valeur
  • Couleur : 5 cartes de même couleur
  • Full House : un brelan + une paire
  • Carré : 4 cartes de même valeur
  • Quinte Flush : une quinte + couleur
  • Quinte Flush Royale : couleur + une suite (10...AS)

Dans une suite, l'AS peut etre considéré comme la plus petite(A-2-3-4-5) ou comme la plus grande (10-J-Q-K-A). Avant de débuter la partie, le joueur doit choisir une mise.Par la suite, le joueur clique sur bouton COMMENCER LA PARTIE afin d'afficher une première série de cartes. S'il le désire, le joueur peut échanger ou garder les cartes de son choix. Pour valider son choix, le joueur doit cliquer sur le bouton ECHANGER CARTES et de nouvelles cartes s'afficheront. Pour terminer la partie, le joueur doit cliquer sur le bouton VALIDER LE JEU, et selon la combinaison obtenu, les gains seront réajustés.

Bonne partie !
